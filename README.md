# OpsBoard
Một cổng Service Desk nội bộ cho doanh nghiệp: người dùng đăng nhập bằng OAuth (Google/GitHub), tạo &amp; theo dõi ticket, bình luận, gán người xử lý, kèm dashboard phân tích (ECharts/Chart.js) về SLA, throughput, lead time, top categories… Ứng dụng SSR SEO tốt, responsive/mobile-first, có i18n (en/vi/ko).
